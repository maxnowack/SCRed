Beschreibung:
	Dieses Programm verteilt Zuhörer auf
	verschiedene SHOUTcast-Streams. Dabei
	wird der Zugriff auf das SHOUTcast DNAS
	Interface blockiert. Dies bedeutet nicht,
	dass man dieses nicht mehr aufrufen kann,
	es wird lediglich der Link nicht angezeigt.

Anforderungen:
	Apache-Webserver mit PHP>=v5.0 & mod_rewrite
	Einen oder mehrere SHOUTcast-Streams
	
Installation:
	1.	Laden Sie die Dateien aus dem ZIP-Archiv
		in ein beliebiges Verzeichnis auf Ihrem Server.
	
	2.	Geben Sie php Schreibrechte auf die Datei "i".
		
	3.	Passen Sie die "config.php" nach Ihren Wünschen an.
		
	4.	Rufen Sie das Verzeichnis auf Ihrem Webserver,
		in das Sie die Dateien hochgeladen haben,
		über einen beliebigen Webbrowser auf.
